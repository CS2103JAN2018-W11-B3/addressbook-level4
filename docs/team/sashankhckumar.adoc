== Overview

Infinity Book (IB) is an address book application for Tech recruiters who prefer to use a desktop app for managing candidates. +

IB provides *end-to-end support*, from searching for candidates,to adding job postings and saving their resumes and interviews.

== Summary of contributions

* *Major enhancement*: added *the ability to add job postings*
** What it does: allows the recruiter to add a job posting along with preferred location, skills, tags and position title.
** Justification: This feature makes all the information in the Infinity Book useful to a recruiter by providing a direct method to match and access relevant data on the Infinity Book for their job.
** Highlights: This enhancement modifies the structure of the model and data handling in the entire application was modified to suit an additional primary data list. Implementation required many changes to the testing tools of the Infinity Book as well.

* *Minor enhancement*: implemented an auto-correct and auto-complete feature that assists users enter the correct commands in the Infinity Book, catching one edit distance corrections.

* *Minor enhancement*: implemented a skills command and introduced an additional skills field for each candidate in the Infinity Book to allows users to store more relevant information about each candidate.
