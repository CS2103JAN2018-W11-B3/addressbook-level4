== PROJECT: Infinity Book

== Overview

Infinity Book is a desktop address book application used for HR management recruiting candidates. The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.

== Summary of contributions

* *Major enhancement*: added the interview management.
** What it does: allows the user to add the interview relevant command which are add interview command,
find interview command, listing interview command.
** Justification: This feature improves the product significantly since the application is for HR management recruiting candidates
, they need to create and manage the list of interviews for candidates.
** Highlights: This enhancement affects existing commands and commands to be added in future. It required an in-depth analysis of design alternatives. The implementation too was challenging as it required changes to existing model, ui, and storage.


* *Minor enhancement*: added the view command.
** What it does: allows user to view a specific person by email ID and display the linked page of the person.
** Highlights: This enhancement affects existing commands and commands to be added in future. It required an in-depth analysis of design alternatives. The implementation too was challenging as it required changes to existing commands.
